# Лабораторная работа №3: Операции с матрицами и перегрузка операторов

**Вариант:** Matrix Calculator System

## Цель работы
Изучение продвинутых принципов ООП в C#: перегрузка операторов, пользовательские исключения и паттерн "Прототип". Реализация инструмента для работы с квадратными матрицами.

## Структура проекта
- **Program.cs** – точка входа, демонстрация работы калькулятора, обработка исключений.
- **lab3.csproj** – файл проекта для .NET.
- **README.md** – описание проекта.

## Стандарты кодирования
- **Отступы** – строго 2 пробела.
- **Именование** – `camelCase` для локальных переменных.
- **Структура методов** – разделение на блоки объявления переменных и вычислений.
- **Обработка ошибок** – пользовательские исключения для несовместимых размеров и вырожденных матриц.
- **Клонирование** – реализация интерфейса `IMyCloneable` (глубокое копирование).

## Ключевые возможности
- Перегрузка операторов: `+`, `*`, `>`, `<`, `>=`, `<=`, `==`, `!=`, `true`/`false`, явное приведение к `double`.
- Вычисление определителя (рекурсивно через миноры).
- Нахождение обратной матрицы.
- Сравнение матриц по определителю (`IComparable<T>`).
- Глубокое клонирование.
- Пользовательские исключения: `MatrixException`, `SizeMismatchException`, `SingularMatrixException`.

## Запуск
1. Убедитесь, что установлен .NET SDK (совместимый с .NET Framework 4.7.2 или новее).
2. Откройте терминал в папке проекта:
   ```powershell
   cd lab_03
   ```
3. Запустите программу:
   ```powershell
   dotnet run
   ```
4. Введите размер матрицы и наблюдайте демонстрацию работы.
